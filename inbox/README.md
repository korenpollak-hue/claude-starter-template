# Inbox

Hier kommen neue, unbearbeitete Dateien rein — bevor du weißt, wo sie hingehören.

**Workflow:**
1. Neue Dateien → hierher ablegen
2. Regelmäßig sortieren → in den richtigen `workspace/`-Ordner verschieben
3. Inbox leer halten

Dieses System kommt aus dem GTD-Prinzip (Getting Things Done).
