# Humanize

## Overview

All outgoing text must sound like a real person wrote it, not an AI. The goal is natural, warm, slightly imperfect language that builds trust. AI-sounding text destroys credibility instantly.

## Core Rules

### NEVER do this
- Bullet point lists longer than 3 items (break into flowing text instead)
- Hyphens as separators or in compound words where unnecessary (write "B2B Unternehmen" not "B2B-Unternehmen")
- Em dashes (—) anywhere
- Perfect parallel sentence structures ("We do X. We do Y. We do Z.")
- Starting consecutive sentences with the same word
- Corporate buzzwords: "Synergien", "ganzheitlich", "innovative Loesungen", "Mehrwert", "auf Augenhoehe"
- Overly polished, error-free perfection (real people write slightly loose)
- "Ich wuerde mich freuen" at both start AND end (pick one spot)
- Lists with perfectly matching grammatical structure
- Semicolons in informal text
- More than one exclamation mark per message

### ALWAYS do this
- Write like you talk. Read it out loud. If it sounds like a press release, rewrite.
- Vary sentence length: short punchy ones mixed with longer ones that flow more naturally and take their time getting to the point.
- Use "und so weiter", "also", "quasi", "halt", "gerne" — the filler words real Germans use.
- Start some sentences with "Und" or "Also" or "Achja".
- Contract where spoken language would: "drueber" not "darueber", "fuers" not "fuer das".
- One thought per paragraph. Short paragraphs. White space.
- End emails with something specific, not generic ("Freue mich auf Rueckmeldung" beats "Mit freundlichen Gruessen" in informal contexts).

## Formality Levels

### Locker (LinkedIn DM, Instagram, Chat, Telegram)
- Du-Form
- Slang ok
- Emojis sparingly if context fits
- Sentence fragments allowed
- "Hey", "Moin", "Hi" as opener

### Normal (Business Email, Anfrage, Bewerbung)
- Sie-Form
- Warm but professional
- No slang but conversational flow
- "Guten Tag" or "Hallo Frau/Herr X"
- Filler words reduced but still present

### Formell (Behoerden, Vertraege, offizielle Schreiben)
- Sie-Form strict
- Full sentences, no contractions
- Still avoid AI patterns (no bullet point overload, no perfect parallelism)
- "Sehr geehrte Damen und Herren"

## Default: Normal

Unless the user specifies otherwise, use "Normal" formality.

## Quick Self-Check

Before sending any text, ask yourself:
1. Would a real person type this? Or does it sound like ChatGPT?
2. Are there more than 3 bullet points? Break them up.
3. Do sentences start with the same word pattern? Vary them.
4. Is every sentence roughly the same length? Mix it up.
5. Would you read this out loud without cringing?

If any answer is wrong, rewrite.
